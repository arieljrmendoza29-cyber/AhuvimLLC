# AHUVIM LLC — Welcome Home

**Rental Management Services Website**

Professional website for AHUVIM LLC, a rental management company based in Chesapeake, Virginia. Established August 23, 2024.

## Overview

A fully responsive, single-page website featuring:
- Hero section with background video
- About Us with company stats
- Services showcase (6 core services)
- Mission & Vision with embedded videos
- Why Us differentiators
- Client testimonials
- Vision in Motion video gallery
- Contact form and company information

## Tech Stack

- **HTML5** — Semantic markup
- **CSS3** — Custom properties, Grid, Flexbox, animations
- **Vanilla JavaScript** — Scroll effects, intersection observers, form handling
- **Google Fonts** — Playfair Display + DM Sans

No build tools or frameworks required.

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/ahuvimllc-website.git
   ```
2. Open `index.html` in your browser — no server required.

For local development with live reload, use any static file server:
```bash
npx serve .
```

## Deployment

This is a static site and can be deployed to:
- **GitHub Pages** — Push to `main`, enable Pages in Settings
- **Netlify** — Drag and drop the folder
- **Vercel** — Import the repository

## Responsive Breakpoints

- Desktop: 1200px+
- Tablet: 768px – 1024px
- Mobile: 480px – 768px
- Small mobile: < 480px
